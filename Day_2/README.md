## ✔ Topics Covered

### **1. Strings**

* Strings store text.
* Created using:

  * Single quotes `' '`
  * Double quotes " "
  * Triple quotes `""" """`
* Escape sequences:

  * `
    ` → new line
  * `	` → tab space

### **2. String Operations**

* **Concatenation**: joining strings using `+`
* **Length**: `len()` function
* **Indexing**: accessing characters by position
* **Slicing**: extracting parts of a string

### **3. String Functions**

* `.endswith()`
* `.capitalize()`
* `.replace()`
* `.find()`
* `.count()`
